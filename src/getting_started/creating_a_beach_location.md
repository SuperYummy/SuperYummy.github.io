## Creating A Beach Location
Worlds are location based. Everything in your world begins with a location

1. Enter the command **/world**

    <img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/1.png"/>

   *The user is entering /world in their Discord chat bar*

2. At the bottom of the message, find the **Collections** section and press its button

    <img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/2.png"/>

   *The world message with a Collections section*

3. Collections contain the data that makes up your world. Turn the page until you find the Locations section

    <img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/3.png"/>

   *The first page of Collections*

4. Press the **Location** buttons

    <img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/4.png"/>

   *The Locations section appearing on page 3*

5. Locations are places where things can exist. To create your world's first location, press that **Create** button

    <img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/5.png"/>

   *The Locations message with a Create button*

    <img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/6.png"/>

   *The new channel beachðŸŒŠ*

7. In the Creation message, select the beachðŸŒŠ channel

    <img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/7.png"/>

   *Selecting the beachðŸŒŠ channel*

Selecting the channel creates a default location. It will also create a message in that channel that automatically displays the location's information. If the channel becaomse quiet, Chiaki will resend this message

<img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/7_1.png"/>

*A created default location*

<img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/7_2.png"/>

*The location message in the beachðŸŒŠ channel*

8. Rename the default location to Beach

9. Update your location image so that it looks like the beach.

10. (Optional) Could add a description too


<img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/end_1.png"/>

*The location renamed to Beach, with a beach image and description: "Ahh~ the breeze"

<img src="https://raw.githubusercontent.com/SuperYummy/SuperYummy.github.io/refs/heads/main/images/Creating%20A%20Beach%20Location/end_2.png"/>
*The updated location message in the beachðŸŒŠ channel*

Done! You have created your world's first location, the Beach!
